# Red team 2026-07-10 — full-change adversarial pass + performance/glide audit

Four adversarial agents over every change since 2026-07-09 (130 commits, ~16k insertions),
plus a live perf measurement (ui/js frame samplers) and a sheet-glide composite audit.
Owner directives: uncompromising ideal-shape; 60fps UI+JS across ALL transitions; ONE sheet
gliding between snap points across scene/page changes (both directions, press-up timing,
especially back-out).

## P1 findings (fix now)

- **[RT-1] Deep-link crash on malformed percent-encoding.** desire-url-codec.ts decodes
  segments with bare decodeURIComponent; `crave://q/50%` throws in the Linking listener
  (crash) / silently kills cold-launch intents. FIX: safeDecode chokepoint in the codec.
- **[RT-2] isTransitionAnimating is permanently false — the content freeze died with the
  machine.** profile-direct-presentation-runtime always writes 'open'; nothing writes
  'opening'/'closing'. shouldFreezeRestaurantPanelContent can never be true → hydration
  mid-slide hard-swaps panel content (the exact jank the freeze prevented). FIX: re-feed
  the freeze from the PF in-flight signal (the L3 slice-1 promise), retire transitionStatus
  to the shape its one real reader needs.
- **[RT-3] EditProfile save deadlock.** Parallel claimUsername+updateMe: claim succeeds,
  update fails → retry re-claims own name → 30-day cooldown 400 → display name unsavable
  until remount. FIX: sequence ops, reconcile local state per-op.
- **[RT-4] The notifications migration bundles DESTRUCTIVE drift**: DROP of the load-bearing
  HNSW index, DROP TYPE checkout_session_status, photos FK churn — swept in by migrate dev.
  A prod migrate deploy would degrade search until boot self-heal. FIX: excise drift from
  the migration (prod has not applied it), fix local checksum.
- **[RT-5] Duplicate reconnect auto-retry lane** (use-search-foreground-command-runtime:60):
  tuple-equal no-op for natural sessions AND coerces shortcut/entity sessions into a bogus
  natural LLM search racing the policy-domain retry. FIX: delete the lane; retry has one
  owner (retrySearchDesiredResolution).
- **[RT-6] Pending request-decoration leak on tuple-equal submits** (single global slot,
  consumed by the NEXT dispatch of any class → stale entity wire fields ride a later
  filter rerun). FIX: writer takes the decoration with the write; cleared on changed:false.
- **[RT-7] Representative-first group order is LOST before the engine** (map). The flag
  lives on the catalog entry only; the controller drops it, the visual collect re-sorts by
  score→markerKey (coordinate-lexicographic within a group), and the native sort has no
  tiebreaker (Swift sort not stable). A sibling can win the group's slot — pin+badge at the
  wrong location. FIX: carry isGroupRepresentative into feature.properties, tiebreak the
  collect sort, deterministic native tiebreak.
